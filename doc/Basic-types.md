Supported basic types are:

| Pack type     | C++ type      | Default value |
| ------------- | ------------- | ------------- |
| Int32         | int32_t       | 0             |
| Int64         | int64_t       | 0             |
| UInt32        | uint32_t      | 0             |
| UInt64        | uint64_t      | 0             |
| Float         | float         | 0.0           |
| Double        | double        | 0.0           |
| Bool          | bool          | false         |
| String        | std::string   | ""            |
| UChar         | unsigned char | '\0'          |
